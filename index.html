            Gatherfield
          

          

            Gatherfield is a Markdown workspace for writing that spreads across notes, drafts, research, sources, outlines, and revisions. Keep the text in plain files, follow threads without losing your place, and see how the pieces relate as a map.
          

        

      


      

        

          
Writing with structure

          
Keep your notes connected.

          

            Long projects sprawl into notes, sources, outlines, and half-finished drafts. Gatherfield keeps those pieces editable as text while the map, references, annotations, history, and read-through stay close to the pages that use them.
          

        


        

          

            
01 / Note links

            
Your links become a visual map.

            

              Start with an empty map, create the first note, then link another. Gatherfield opens it beside the original and grows the map from the links in your text.
            

          

          
        


        

          

            
02 / Arranged map

            
Take over when auto layout is not what you meant.

            

              Auto layout shows what is connected. Manual layout lets you drag notes into a readable order, then keep that arrangement with the note.
            

          

          
        


        

          

            
03 / Alternate drafts

            
Experiment with drafts and the paths they create.

            

              Save alternate versions when an idea branches. Switch drafts and the map follows the links in the version you chose.
            

          

          
        


        

          

            
04 / Draft history

            
Revise freely.

            

              Follow an idea too far, then pull back. History restores an earlier draft while the note remains a normal text file.
            

          

          
        


        

          

            
05 / Annotations

            
Leave comments beside the text, not inside it.

            

              Highlight a line, leave a comment, and come back to it later. The annotation stays attached to the passage while the underlying text stays clean.
            

          

          
        


        

          

            
06 / Read-through

            
Keep a read-through current without copy-paste.

            

              Write and revise in separate linked notes. Draft follows the flow, so moved, added, or changed notes update the read-through.
            

          

          
        


        

          

            
07 / Passages and notes

            
Change a note's scale as the work changes.

            

              Start with the shape that works now. If a passage needs room, make it a note; if a linked note belongs inline, bring it back into the text.
            

          

          
        

      


      

        

          
Built for long-lived projects

          
Your notes stay ordinary text files.

          

            Gatherfield can be the main workspace, or the layer you bring in when a project needs mapping, sources, versions, and read-throughs. The notes stay readable as plain Markdown files.
          

        

        

          

            Shared context
            
The page, map, files, and references stay together.

            
Move through a project without mentally reconciling separate outlines, source lists, and diagrams.

          

          

            Dockable panes
            
Set up the room for the writing task.

            
Put the draft, map, files, and research where the current pass needs them.

          

          

            Optional metadata
            
Style the map only when it helps.

            
Plaintext remains the body. Frontmatter can add graph presentation details for notes that need them.

          

          

            Sources
            
Attach references without adding them to the map.

            
Keep citations, source notes, and research close to the draft while the visual map stays focused on the notes you are connecting.
